## Отчет по лабораторной работе "Разработка инфраструктуры MVP AI приложения"  
**Цель работы:**  
Создать прототип AI-приложения с базовой функциональностью.  

**Ход работы:**    
1. Придумала идею приложения - AI-сервис для автоматической организации портфолио фотографа.  Сервис анализирует изображения с помощью Google Cloud Vision API и распределяет их по категориям (например: свадьба, портрет, концерт), что упрощает управление большим количеством фотографий.
 
2. Описание работы сервиса:     

**Стадия 1 - начальное (MVP)**  
Сервис рассчитан на небольшое количество пользователей (до 100).  
Используется минимальный набор облачных сервисов:  
   - Cloud Run — размещение приложения  
   - Cloud Storage — хранение изображений  
   - Cloud Vision API — анализ изображений
   
**Стадия 2 - тестирование**  
Количество пользователей увеличивается (до 1000), появляются требования к стабильности.  
Добавляется:  
   - Cloud Load Balancing — распределение нагрузки  
   - авто-масштабирование Cloud Run

**Стадия 3 - прод (масштабирование)**  
Сервис используется большим количеством пользователей.  
Добавляются:  
   - Cloud CDN — ускорение загрузки изображений
   - Cloud SQL - позволяет сохранять теги, искать/фильтровать фото
   - резервное копирование БД

3. Подсчет расходов:

| Ресурс | Обоснование | Расходы|  
|-----------|-----------|-----------|  
| Cloud run | Управление серверами | ~$20/мес |  
| Cloud storage | Хранение изображений | 1,9$/100gb |  
| Cloud Vision API| AI-анализ фото | 2,25$/2000 фото (меньше 1000 фото - бесплатно) |  
| Cloud Load Balancing| Балансировка нагрузки - распределяет запросы между серверами | ~$20/мес |  
| Cloud CDN| Ускорение загрузки контента | Зависит от трафика (~30$/500gb) |  
| Cloud SQL| Поиск и структурирование файлов | ~115$/мес |  


Для запуска целесообразно выбрать начальную стадию, на старте будет ограниченное количество пользователей и меньше стоимость. При масштабировании расходы на сервисы будут расти, так как становится больше пользователей и объем загруженных снимков увеличивается (= растет нагрузка).   
> Оценка стоимости выполнена на основе официального калькулятора Google Cloud Pricing Calculator.
